你现在看到的博客由 Hugo（静态站点生成器）+ PaperMod（主题）驱动，使用 GitHub Actions 自动部署到 GitHub Pages。

- Markdown 支持表格、代码块、公式、脚注等
- 代码块自动高亮，并带一键复制按钮
- 主题内置暗黑模式、阅读时长、社交分享、RSS、站内搜索
